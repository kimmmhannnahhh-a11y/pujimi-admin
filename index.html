📊 공시지원금 업데이트 (요금계산기)

아래 초록 버튼을 브라우저 즐겨찾기 바에 한 번 드래그해 두세요. 그다음부터는 제로노트에 로그인한 상태에서 그 즐겨찾기를 누르면 → 전 기종·전 통신사·요금제별 공시지원금이 자동으로 크롤링돼서 요금계산기에 바로 반영돼요.
📲 공시 업데이트
↑ 이 버튼을 즐겨찾기 바로 드래그
  1. 위 📲 공시 업데이트 버튼을 브라우저 즐겨찾기 바로 드래그 (최초 1회만)
  2. 제로노트(xeronote.co.kr) 접속 → 로그인
  3. 즐겨찾기에 추가된 📲 공시 업데이트 클릭
  4. 우하단에 "동기화 완료!" 뜨면 끝 → 요금계산기 새로고침
